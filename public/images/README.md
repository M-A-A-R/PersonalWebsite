# Images

Add your images here. The site expects:

**Photo gallery** (horizontal strip near top):
- `gallery-1.jpg` — main/larger image
- `gallery-2.jpg` — thumbnail
- `gallery-3.jpg` — thumbnail

**Photography section** (your work):
- `photography-1.jpg`
- `photography-2.jpg`
- `photography-3.jpg`

Edit `CONTENT.md` and `src/data.ts` to add more images or change paths.
